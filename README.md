## About The Project

The **Passport Photo Resizer** is a free tool designed to solve a common problem: creating compliant passport or visa photos without compromising privacy or paying exorbitant fees. Traditional photo booths are costly, and online services often require you to upload your sensitive biometric data to their servers.

This tool leverages the power of the browser's HTML Canvas API to perform all image processingâ€”loading, cropping, and resizingâ€”directly on the user's device. Your photos are never sent over the network, offering absolute peace of mind.

This project is built with a modern, robust, and privacy-focused tech stack:

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js (version 16 or later) and a package manager installed.
-   `npm`
    ```sh
    npm install npm@latest -g
    ```

### Installation & Running

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hsinidev/passport-photo-resizer.git
    cd passport-photo-resizer
    ```

2.  **Install NPM packages:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
